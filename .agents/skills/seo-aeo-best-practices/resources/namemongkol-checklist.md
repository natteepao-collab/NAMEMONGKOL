# NameMongkol Technical SEO & Content Checklist

Derived from the 30-Day SEO Strategy.

## 1. Article Structure

- **H1 Heading**: MUST contain the Primary Keyword (e.g., "วิเคราะห์ชื่อมงคล...").
- **Introduction**: 
  - Hook the reader in the first **100 words**.
  - Include the Primary Keyword naturally.
- **Internal Linking**:
  - **Mandatory**: Link to `/name-analysis` (Analyze your name now for free).
  - **Related**: Link to other relevant articles (e.g., "See result of number 45" -> "Money numbers").
  - **CTA**: End every post with: "อยากรู้ว่าชื่อของคุณดีแค่ไหน? [คลิกวิเคราะห์ฟรีที่นี่](/name-analysis)"

## 2. Image Optimization

- **File Name**: Use English keywords separated by hyphens.
  - ✅ `auspicious-name-meaning.jpg`
  - ❌ `IMG_001.jpg`, `ชื่อมงคล.jpg`
- **Alt Text**: Describe the image using keywords in Thai.
  - Example: "ตารางทักษาปกรณ์สำหรับคนเกิดวันจันทร์"

## 3. Distribution Channels (Off-Page)

- **Facebook**: Post link with catchy caption + attractive image.
- **Line OA**: Broadcast weekly summary (e.g., "Weekly Horoscope") linking to articles.
- **TikTok**: Create short clips summarizing the topic (e.g., "3 banned letters") and link in bio.
